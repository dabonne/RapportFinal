\chapter*{Résumé}
\addcontentsline{toc}{chapter}{Résumé}%Pour l'ajout dans la table des matières au même rang que chapitre

Ce memoire traite de la segmentation geometrique et photometrique d’images ae-
riènnes acquisent par drones en vue d’annoté les images de carriere. Nous y proposons
deux solutons.
La premiere s’apuie sur les CNN pour ameliorer et les algorithmes de classification
d’images pour ameliorer les resultats d’un première etude basée exclusivement sur
la segmentation geometrique.
Dans la seconde solution, nous utilisons conjointement les informations geometriques
et photometrique pour annoter les carrières. Dans ce cas nous utilisons les CNN et
les algorithmes de segmentation sementiques.
Nous y présentons également des résultats d’application en utilisant ces deux solu-
tions. \\

\textbf{Mots clés} : Deep Learning ; CNN ; Segmentation sementiques ; Tensorflow ; Ke-
ras Images acquisent par drones ; classification
